# 10-Meter-Air-Rifle-Rahmad-Habib-Syakban
10 Meter Air Rifle is a precision shooting sport that originated in Europe in the late 19th century and later became part of the Olympic Games in 1984. The sport emphasizes focus, discipline, and technical accuracy.At Universitas Negeri Jakarta (UNJ), 10m Air Rifle is developed as a competitive student sport, supporting athlete development
